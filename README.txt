HOW TO HOST ON GITHUB PAGES
1. Create a new GitHub repository.
2. Upload every file and folder inside this ZIP to the repository root.
3. Go to Settings > Pages.
4. Under Branch, choose main and /root, then Save.
5. Open the GitHub Pages link.
6. Navigate to scratch.html.
7. Submit that scratch.html URL in the assignment.

The URL should look like:
https://YOURUSERNAME.github.io/YOURREPOSITORY/scratch.html
